## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/NiksheyYadav/Dyple.git
cd Dyple
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

1. **Configure Network Architecture** - Adjust the number of layers, neurons per layer, and activation functions
2. **Set Optimizer Parameters** - Choose between SGD and Adam, adjust learning rate and other hyperparameters
3. **Add Training Data** - Input your training examples with inputs and target outputs
4. **Train** - Click the Train button to start the training process
5. **Visualize** - Watch real-time updates of network activations, weight distributions, and loss curves
